Funcionalidades Obrigatórias

1. Gestão de Clientes

  • Listar clientes (tabela com Nome, CPF, Email)
  • Criar novo cliente (Nome, CPF, Email)
  • Editar cliente
  • Deletar cliente
  • Validar: campos vazios, email com @

2. Gestão de Contas

  • Listar contas (Número, Cliente, Tipo, Saldo)
  • Criar conta (selecionar cliente, tipo: Corrente/Poupança, saldo inicial 0)
  • Editar status da conta (Ativa/Inativa)
  • Deletar conta

3. Transações

  • Registrar depósito (conta, valor, atualizar saldo)
  • Registrar saque (conta, valor, validar saldo suficiente)
  • Listar histórico de transações (Data, Tipo, Valor, Saldo Novo)
